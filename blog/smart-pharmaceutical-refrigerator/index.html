1. INTRODUCTION

Nowadays most of the pharmaceutical stores have refrigerators to store some vaccines. Some of the vaccines inside the refrigerator might get exhausted very soon due to high demand, which may lead to delay in replenishing of the stock in time because at present there is no automated system to indicate the reduction in the number of vaccines. There are chances of misplacing the vaccine into different compartment unknowingly. Sometimes people forget to buy the vaccine before it gets over. There is no existing solution which overcomes the abovementioned problems. The proposed solution consists of an IoT system which is capable of: • Monitoring the stock inside the refrigerator, placing an order to the distributor when any vaccine reaches below a specified number and notifying the same to the shopkeeper. • Triggers an alarm when the vaccine is not placed in the appropriate compartment. • Notifies the customer about reordering the vaccine based on their regular usage. We are going to use a microcontroller board which continuously monitors the stock inside the refrigerator by integrating with several sensors and places an order to the distributor using a WiFi module to avoid logistic delay. Using image processing technique an alarm will trigger if any vaccine is misplaced. Customer’s purchase and usage will be stored in order to compute, when the stock needs to be reordered and notifies the same to the customer.

6. SYSTEM IMPLEMENTATION

For the implementation of this system, certain tools are used – OpenCV (library of programming functions mainly aimed at real-time computer vision), TensorFlowLibrary (free and opensource software library for dataflow and differential programming across a range of tasks)The Ultrasonic sensor will determine the number of vaccines that are present in a compartment of the pharmaceutical refrigerator by measuring the distance from the vaccines. Ultrasonic sensors work by sending out a sound wave at a frequency above the range of human hearing. The transducer of the sensor acts as a microphone to receive and send the ultrasonic sound. The sensor determines the distance to a target bymeasuring time lapses between the sending and receiving of the ultrasonic pulse. The number of objects is said to be low if the distance to the objects is high and using this principle we can identify the number of vaccines that are present in the refrigerator. When the number of vaccines become lesser than threshold then using the Wi-Fi module it is possible to notify vaccine distributor using email. In order to send a mail IFTTT web-based service has been used.
